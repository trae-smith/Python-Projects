# Python Projects

Python programming work completed during my Bachelor of Mathematics 
(Statistics) at Carleton University, covering image processing, 
pixel manipulation, and applied programming.

## Highlighted Projects

### Image Processing — RGB Decomposition & Reconstruction
`comp1005_f22_101211905_assignment_05.py`
Performs pixel-level RGB channel analysis on any image. Calculates 
colour intensity thresholds per pixel and reconstructs the image 
using randomized circle placement scaled by RGB values. Produces 
a stylized output saved to file.
**Tools:** Python, pygame

### Interactive Colour Transformation
`comp1005_f22_101211905_assignment_07.py`
Click-based image editor that applies colour inversion across a 
radius around the cursor position in real time.
**Tools:** Python, pygame
